== newsdict.io : Correct the conntents from Internet.

include::../../badges.adoc[]

newsdict.ioはフォロワーがシェアしている記事などのウェブコンテンツを
クローリングしフィードとして閲覧できるようにします。Railsで作ってあるため拡張が容易です。

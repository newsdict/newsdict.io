== Getting Start

=== 1. Install docker
https://docs.docker.com/engine/install/ubuntu/

=== 2. Install docker-compose
https://docs.docker.com/compose/install/

=== 3. .env
1. copy `.env-example` file to `.env` file.

=== 4. Start development server
```
$ docker/start
```
